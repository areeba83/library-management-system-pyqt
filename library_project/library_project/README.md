# 📚 Library Management System (PyQt5)

A desktop GUI application built using Python and PyQt5 to manage books in a library.

## 🚀 Features
• Add, update and delete books  
• Search books instantly  
• Show availability status  
• Fetch motivational quotes using API  
• Clean GUI interface  

## 🛠 Technologies Used
Python  
PyQt5  
Requests API  

## ▶️ How to Run
pip install -r requirements.txt  
python main.py
